# Store-analysis-and-product-performace
This project analyzes sales, customer behavior, product performance, and store efficiency using Power BI.
This project is an interactive Power BI dashboard built to analyze sales performance, customer behavior, product trends, and store efficiency.

The goal is to transform raw transactional data into meaningful KPIs and business insights that support data-driven decision-making.
The dashboard answers key business questions:

How is total revenue performing over time?

Which stores generate the highest sales?

What is the Average Order Value (AOV)?

What percentage of customers are repeat buyers?

Are deliveries happening on time?

Which products are low in stock?
